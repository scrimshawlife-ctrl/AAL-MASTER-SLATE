# PsyFi

## Overview
PsyFi is a user-facing interface for introspection, symbolic mapping, and emotional navigation.

## Purpose
- Provide tools for self-examination using symbolic structures.
- Surface patterns and tensions in a user's narratives, habits, and environments.
- Integrate audio and dream data through BeatOven and Noctis Engine.

## Architecture
- Input UI for journaling and prompts
- Symbolic tagging and clustering engine
- Visualization and mapping layer
- Integration with Abraxas and Noctis frames

## IO Model
- Inputs: text entries, responses to prompts, optional metadata.
- Outputs: symbolic maps, vectors, and guidance recommendations.

## Relationship to AAL-Core
PsyFi is a front-end facing module that primarily produces user-generated ResonanceFrames.

## Example Output (Text)
- "You are currently in a 'threshold' phase with high tension around autonomy vs. obligation."
